# Tutorials Overview

Welcome to the ACoord tutorials! These step-by-step guides will help you master the key features of ACoord.

## Available Tutorials

### [Viewing Structures](/tutorials/viewing-structures)

Learn how to:
- Open structure files in ACoord
- Navigate the 3D view (rotate, pan, zoom)
- Use different rendering styles
- Adjust lighting and display settings

### [Editing Atoms](/tutorials/editing-atoms)

Learn how to:
- Add new atoms to a structure
- Delete selected atoms
- Move atoms by dragging
- Copy and paste atoms
- Change atom colors and labels

### [Working with Trajectories](/tutorials/working-with-trajectories)

Learn how to:
- Open trajectory files (XDATCAR, etc.)
- Navigate between frames
- Play and pause animations
- Export specific frames

## Recommended Learning Path

1. **New to ACoord?** Start with [Getting Started](/guide/getting-started)
2. **First structure?** Follow [Viewing Structures](/tutorials/viewing-structures)
3. **Want to edit?** Continue with [Editing Atoms](/tutorials/editing-atoms)
4. **Working with MD?** Check out [Trajectories](/tutorials/working-with-trajectories)

## Quick Reference

| Task | Action |
|------|--------|
| Rotate view | Left-click + drag |
| Pan view | Right-click + drag |
| Zoom | Scroll wheel |
| Select atom | Click on atom |
| Multi-select | Shift + click |
| Delete atom | Select + Delete key |
| Move atom | Drag atom |
| Undo | Ctrl+Z |
| Redo | Ctrl+Y |
